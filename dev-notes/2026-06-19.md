# 2026-06-19 开发小记

## 今日改动

### 1. 详情页合并重构

**问题**: 歌单详情 (`playlist.ts`) 和专辑详情 (`album.ts`) 是两个独立页面，但布局和逻辑高度相似。

**方案**: 合并为 `detail.ts`，导出 `renderCollectionDetail(container, type, id)`。

**差异点**:
| | 歌单 | 专辑 |
|---|---|---|
| API | `playlistDetail` + `playlistTrackAll` | `album`（直接返回歌曲） |
| 封面 | `coverImgUrl` | `picUrl` |
| 元数据 | 创建者 · 曲目数 | 艺人 · 发行日期 · 曲目数 |
| 描述 | `description` | `description` |
| 额外 | — | `company`（厂牌） |

**统一接口**:
```ts
interface CollectionMeta {
  name: string;
  coverUrl: string;
  description: string;
  subtitle: string;   // 已拼好的元数据文本
  tracks: Track[];
}
```

### 2. 路由注册
